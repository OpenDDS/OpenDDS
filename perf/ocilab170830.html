OpenDDS - Performance Testing Environments

OCILab 2017

The test environment includes the test hosts and the network equipment connecting the hosts. All hosts in the tests were the same and a single network was used to connect them.
  1. Test Hosts
    • HP Compaq 6000 PRO
    • Intel(R) Core(TM)2 Duo CPU E8400 3.00GHz
    • 6144 KB cache
    • 5.5 GB RAM
    • Intel Corporation 82567LM-3 Gigabit Network Connection (rev 02)
    • CentOS 7, Linux kernel 3.10.0-229.el7.x86_64 SMP
  2. Test Network
    • HP J4903A ProCurve Switch 2824
